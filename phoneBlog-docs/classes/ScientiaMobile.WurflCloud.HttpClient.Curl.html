setProxy()

setProxy(string  $proxy)

Use a proxy to access the WURFL Cloud service Proxy URL should be in the format: protocol://[user:pass@]ip_or_hostname:port Ex: http://192.168.1.10:8080 socks4://192.168.1.10:8080 socks4a://192.168.1.10:8080 socks5://myuser:mypass@192.168.1.10:8080

This option gets passed straight to CURL and whether or not it works for you depends on your version of PHP/CURL.

Parameters

string $proxy

URL of proxy server
