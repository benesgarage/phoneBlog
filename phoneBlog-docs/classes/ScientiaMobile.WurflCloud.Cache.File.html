\ScientiaMobile\WurflCloud\Cache

\ScientiaMobile\WurflCloud\CacheFile

The FileSystem Cache Provider

An example of using the Filesystem for caching: // Create Configuration object $config = new ScientiaMobile\WurflCloud\Config(); // Set API Key $config->api_key = 'xxxxxx:xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx'; // Use Filesystem Caching $cache = new ScientiaMobile\WurflCloud\Cache\File(); // Create Client $client = new ScientiaMobile\WurflCloud\Client($config, $cache);

You can also specify the cache directory and cache expiration: // Create Cache object $cache = new File(); // Set Cache directory $cache->cache_dir = '/tmp/'; // Set Cache expiration $cache->cache_expiration = 86400;

$cache_expiration

$cache_expiration : Int

Number of seconds to keep device cached in memory. Default: 0 - forever.

Note: the device will eventually be pushed out of memory if the memcached process runs out of memory. Tip: 3600 = 1 hour, 86400 = 1 day, 604800 = 1 week, 2592000 = 1 month (30 days)

Type

Int — Seconds to cache the device in memory

$cache_expiration_rand_max : integer

Used to add randomness to the cache expiration. If this value is 0, no randomness will be added, if it's above 0, a random value between 0.
